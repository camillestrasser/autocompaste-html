Instructions

Dear participant , please read the instructions before the experiment.

You are asked to copy and paste some given input sentences, using two different techniques. (Read the description of techniques below)

The experiment field is divided in two parts:

  • On the left-hand side, you’ll find all the instructions.
    • In the table, you can read where you are in the experiment, for example the number tasks you already did or the technique tested.
    • Under the table, you’ll find the task which is required: which technique you should use and which sentence you have to copy.
    • The first one is the Text Editor, the field where you have to copy and paste the sentence.
    • The other ones are the tabs where you’ll find the sentences that you have to copy.

When you’re done, press the “Next task” button to go through the experiments. You will have to go through 3 blocks of 36 trials each, and you will get a small break between the blocks.

Traditional technique

Find the required sentence in the different tabs, it’s highlighted in yellow. Copy this sentence by using Ctrl+C. Then paste it in the text editor window by using Ctrl+V.

Autocompaste technique

Start to type the required sentence in the text editor. The text editor will propose you different sentences that you can select by using up and down arrows. Confirm by pressing Return button. After that you may have to adjust your sentence.
